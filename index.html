About

I am a Software Engineer with 2+ years of experience building full-stack applications using Java, Spring Boot, and React.js, with a strong focus on RESTful APIs, microservices, and integrating AI/LLM capabilities into backend systems. I hold a Master of Science in Software Engineering from George Mason University, Fairfax, VA, and a Bachelor of Engineering in Computer Science from Osmania University, Hyderabad, India.

Throughout my career, I have worked with programming languages including Java, Python, JavaScript, Kotlin, C, C++, SQL, HTML5, and CSS. I have hands-on experience with frameworks such as Spring Boot, React.js, Angular, Node.js, Next.js, Django, and Flask. I am familiar with databases like MySQL, PostgreSQL, MongoDB, and DynamoDB, and have consistently delivered solutions following Agile and Scrum methodologies.

In my current role as a Software Engineer at Progrite Systems, I built RESTful APIs for loan submission, document uploads, and status tracking backed by SQL. I engineered React.js dashboard components with WebSocket support for real-time updates, and designed a document intelligence service using OpenAI GPT API that reduced manual review effort by an estimated 40%. I also deployed containerized services using Docker and Kubernetes on Microsoft Azure with CI/CD pipelines via Maven and GitHub Actions.

During my time at George Mason University, I served as a Graduate Teaching Assistant for Data Structures and Algorithms in Java, guiding 30+ students to a 90% pass rate and boosting overall academic performance by 25%. As a Graduate Research Assistant, I built data preprocessing pipelines using Python, Django, and Flask, implemented anonymization techniques achieving 90% accuracy in de-identification, and applied regression and classification models using TensorFlow and Scikit-learn to identify behavioral trends in academic survey data.

At Digi Planet Techno Solutions, I developed a geolocation-based lending and borrowing web application using ReactJS and Spring Boot, integrated Twilio API for SMS push notifications leading to a 20% increase in user interactions, and optimized MySQL database schema contributing to a 95% improvement in user-facing performance.

I have led and contributed to projects like TaskMaster, a backend microservice using Node.js, TypeScript, and Kafka deployed on AWS EC2 with DevSecOps principles; a Cloud Enhanced Stock Screener leveraging AWS Lambda, Glue, RDS, S3, and QuickSight for interactive stock trend visualization; and a Full Stack Survey Management System using Angular, Spring Boot, and Kubernetes achieving a 95% improvement in UX.

My technical skill set spans Cloud Technologies including Microsoft Azure, AWS (EC2, S3, Lambda, RDS, Glue, QuickSight, DynamoDB), Kubernetes, and Terraform, as well as ML libraries such as TensorFlow, Scikit-learn, Keras, Pandas, NumPy, PySpark, and NLTK. I also work with tools like Git, Docker, Jenkins, Postman, Selenium, Splunk, Maven, and Gradle. My certifications include Spring MVC, Spring Boot and REST Controllers (2023) and PHP, Java and MySQL Training from IIT Bombay (2018), reflecting my commitment to continuous learning and high standards in software engineering.
